# 📦Package Delivery Tracking System

## Overview
A Python-based package delivery tracking system that allows users to track packages and view their delivery status in real-time.  
This project also includes a CI/CD pipeline using GitHub Actions and Gradle for automated builds and testing.

---

- Track package status (In Transit, Delivered, Pending)
- View package history and updates
-  Unique tracking ID for each package
-  Automated build using Gradle
-  CI/CD pipeline using GitHub Actions
-  Unit testing using Pytest

---

##  Tech Stack
- **Language:** Python  
- **Build Tool:** Gradle  
- **Testing:** Pytest  
- **CI/CD:** GitHub Actions
